# Modes
## Mode: PC - Edit Program Change messages for a preset

You get to the mode by holding the button 9 (Arrow Down) you exit the mode by holding Button 4 (Arrow Up)

- **A** Select PC0
- **B** Select PC1
- **C** Select PC2
- **D** Select PC3
- **Arrow Down** Select PC4

- **1** -1
- **2** +1
- **3** -10
- **4** +10
- **Arrow Up** Save and Exit
